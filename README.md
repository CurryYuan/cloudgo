
# cloudgo
## 框架选择 Martini 
Martini是一个强大为了编写模块化Web应用而生的GO语言框架.
### 功能列表
 - 使用极其简单.
 - 无侵入式的设计.
 - 很好的与其他的Go语言包协同使用.
 - 超赞的路径匹配和路由.
 - 模块化的设计 - 容易插入功能件，也容易将其拔出来.
 - 已有很多的中间件可以直接使用.
 - 框架内已拥有很好的开箱即用的功能支持.
 - 完全兼容*http.HandlerFunc*接口.

## 设计思路
 一个简单的go服务器应用，可以进行路由和返回静态网页
## 运行
>go run main.go -p 8080
